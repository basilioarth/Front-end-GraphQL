## 2. Método logout implementado no Zustand store
```ts
logout: () => {
  set({ 
    user: null, 
    token: null, 
    isAuthenticated: false 
  })
  client.clearStore()   // limpa cache do Apollo
}
```
- Remove usuário e token do localStorage (persist)
- Limpa todo o cache do Apollo Client
- Redireciona imediatamente para `/login`

## 3. Handle de logout no Header
```ts
const logout = useAuthStore(state => state.logout)
const navigate = useNavigate()

const handleLogout = () => {
  logout()
  navigate("/login")
}
```
- Botão de logout agora funciona 100%
